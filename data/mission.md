# PHASE Mission

> PLASMA orchestrates. FLUX executes. SOLID validates.

PHASE is an autonomous multi-agent AI system built for EXOAI-1.
It decomposes complex goals into atomic tasks, routes each to a
specialist worker, validates every result through 3-model consensus,
and synthesises a final answer — all autonomously.

## Core Principles

1. **LLM-First** — every decision flows through an LLM call
2. **Continuity** — the system persists state and survives restarts
3. **Minimalism** — no code that doesn't earn its place
4. **Self-Evolution** — PLASMA improves itself via SOLID-gated commits
